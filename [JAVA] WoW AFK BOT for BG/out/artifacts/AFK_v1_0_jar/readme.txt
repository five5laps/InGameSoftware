					Инструкция
======================================================
1.Запустить файл "START.bat" (.BAT)
2.Открыть окно игры
3.Оставить программу работать
======================================================


					Чтобы выключить
======================================================
1.Закрыть файл "START.bat" (.BAT) | Черное окно(windows cmd)
2.Если возникнут проблемы пишите -> https://vk.com/remornia <- (В адресной строке вашего браузера(интернета))
======================================================


					Если не работает
======================================================
1.Проверить запускаете ли вы "START.bat" вместо "AFK_V1.0.jar"
2.Проверить наличие JAVA на вашем ПК(ноутбуке)
3.Если его нет, скачать можно тут -> https://www.java.com/ru/ <-
4.Перезагрузить ПК(ноутбук)
======================================================
